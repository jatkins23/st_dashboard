This contains the 4 components of the search tab, separated into modules
- **SearchForm**: The primary way that users interact with the search functionality. Enter in details necessary for search, including selecting a `location`
- **MapPanel**: Primary screen component. Initially displays past and ongoing projects. After Search, it displays search results
- **DetailsPanel**: Displays details about selected `location`
- **ResultsPanel**: Displays results of the search
